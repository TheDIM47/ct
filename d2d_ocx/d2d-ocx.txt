D2DAXCtrl
=========

   D2DAXCtrl (DAX) is well known Clarion Dat to Dbf converter (D2D)
http://juliasoft.chat.ru/clarion.html modified for use as ActiveX
component inside your application. 

Methods
=======

  Go(SourceFile, DestinationFile)
    SourceFile - path and name of Clarion DAT file (example: "C:\myfile.dat")
    DestinationFile - path and name of target dBASE file ("C:\myfile.dbf")

Properties
==========
